Install gettext:
    Run:
        apt install gettext
    If not working, run:
        sudo apt install gettext

In the root directory backend, add file .env that contains:
    ENABLE_EMAIL_VERIFICATION="false"
    ENABLE_PHONE_NUMBER_VERIFICATION="false"
    DB_CONTAINER_EXTERNAL_PORT=5434
    DB_CONTAINER_INTERNAL_PORT=5432
    DB_IP=postgres
    DB_NAME=leaguer
    DB_ROOT_PASSWORD=root_password
    DB_USER_NM=db_user
    DB_USER_PW=db_password
    DJANGO_CONTAINER_EXTERNAL_PORT=8080
    DJANGO_CONTAINER_INTERNAL_PORT=8000
    PIPLINE="development"
    DJANGO_SECRET_KEY="django-SECRET-KEY"
    EMAIL_HOST="smtp.gmail.com"
    EMAIL_PORT=587
    EMAIL_USE_TLS="true"
    EMAIL_HOST_USER = "leaguerdev@gmail.com"
    EMAIL_HOST_PASSWORD = "as_default_keyboard"  # Use an App Password
    DEFAULT_FROM_EMAIL = "leaguerdev@gmail.com"
    WHATSAPP_INSTANCE_ID="id_xxxx"
    WHATSAPP_INSTANCE_TOKEN="xxxx"
    WHATSAPP_INSTANCE_URL="https://waapi.app/api/v1/instances/id_xxxx/client/action/send-message"


Generate .po Files:
    For each app:
        Create locale directory.
        Change working directory to the app
        Run:
            django-admin makemessages -l ar
            django-admin makemessages -l en
            django-admin makemessages -l fr
Generate .mo files:
    For each app:
        After adding translations, run:
            django-admin compilemessages
