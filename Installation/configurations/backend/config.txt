Install gettext:
    Run:
        apt install gettext
    If not working, run:
        sudo apt install gettext
    If not working, run:
        apt update && apt install -y gettext

Generate .po Files:
    For each app:
        Create locale directory.
        Change working directory to the app
        Run:
            django-admin makemessages -l ar
            django-admin makemessages -l en
            django-admin makemessages -l fr
Generate .mo files:
    For each app:
        After adding translations, run:
            django-admin compilemessages


Firebase firebase-service-account.json:
    Get firebase-service-account.json from Firebase Console
        Go to Firebase Console:  https://console.firebase.google.com/
        Select Your Project.
        Go to Project Settings:
            Click the ⚙️ gear icon in the left sidebar.
            Select Project settings.
        Service Accounts Tab:
            Click on the Service accounts tab.
            Under "Firebase Admin SDK", click Generate new private key. This will download a JSON file (firebase-service-account.json).
            Move the JSON file to your Django project: /your-django-project/firebase-service-account.json
        DO NOT commit this file to GitHub or version control for security reasons.
